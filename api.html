Desarrollo del portafolio

Desarrollo

Para el desarrollo de esta proyecto me propuse realizar mi propio servicio web creado en Node.js y MySQL usando el ORM de Sequelize para el almacenamientos de los datos, y manejo de la misma contemplando las diferentes tablas y relaciones que existen entre ellas.
Para ello realize varios endpoint donde se mostrara el resultado en formato JSON.

En la parte del front-end se desarrollo con React.js por tener una curva de aprendizaje efeciente en donde hago uso de los Hook como useState,useEffect para el menejo de los datos a traves de los componentes utilizando los props.
